# Dochazka Generator

## Что это
Скрипт для автоматической генерации книги посещаемости (kniha docházky) из таблицы рабочих часов (mzdy) для кофейни.

## Проблема
Каждый месяц бухгалтер присылает Excel-файл (Mzdy), где указано, сколько часов каждый сотрудник отработал в каждый день. Из этого файла нужно вручную составить второй Excel-файл — книгу посещаемости (Docházka) с конкретным временем прихода и ухода каждого сотрудника. Это рутинная работа, которая занимает много времени.

## Решение
Python-скрипт, который:
1. Читает файл Mzdy (часы по дням по сотрудникам)
2. Автоматически распределяет правдоподобные смены внутри рабочих часов кофейни
3. Генерирует файл Docházka с отдельным листом на каждого сотрудника

## Правила распределения смен
- Время открытия: будни 6:30, выходные 8:00 (настраивается в скрипте)
- Закрытие = открытие + часы работы кофейни из столбца B
- 2 сотрудника: один открывает, второй закрывает, перекрытие в середине
- 3+ сотрудников: последовательно друг за другом от открытия
- Конкретное время прихода/ухода не важно — важно, чтобы смены покрывали рабочие часы кофейни без разрывов

## Формат входного файла (Mzdy)
- Имя файла: `Mzdy_MM.YYYY.xlsx` (например `Mzdy_03.2026.xlsx`)
- Лист с дневными данными: в ячейке B1 должно быть слово "часов"
- Строка 1: заголовки — B = часы кофейни, D и далее = фамилии сотрудников
- Строки 5–35: данные по дням (A = номер дня, B = часы кофейни, D+ = часы сотрудников)
- Опционально: второй лист с полными именами (имя + фамилия)

В папке лежит шаблон `Šablona_Mzdy.xlsx` — можно отдать бухгалтеру для стандартизации.
